## Welche Berechtigungen nutzt die App?

### Kamera-Berechtigung
**Warum**: Zum Scannen von Barcodes auf Medikamenten und Produkten.

**Datenschutz**:
- Die Kamera wird nur beim aktiven Scannen verwendet
- Es werden KEINE Fotos gespeichert
- Es werden KEINE Bilder an Server übertragen
- Die Verarbeitung erfolgt nur lokal auf Ihrem Gerät

### Internet-Verbindung
**Warum**: Für die Synchronisierung mit der Datenbank (Supabase, EU-Server).

## Welche Daten werden gespeichert?

### Auf Ihrem Gerät (lokal):
- App-Einstellungen
- Anmeldedaten (verschlüsselt)
- Cache für Offline-Nutzung

Sie können diese Daten jederzeit durch Löschen der App entfernen.

### In der Cloud (Supabase, EU):
- Ihr Benutzerkonto (E-Mail, verschlüsseltes Passwort)
- Von Ihnen eingegebene Gesundheitsdaten (Medikamente, Übungen, Termine)
- Ihre Notizen und Einträge

**Sicherheit**:
- Verschlüsselte Übertragung (HTTPS)
- Server in der EU (DSGVO-konform)
- Sichere Authentifizierung

## Werden Daten weitergegeben?

**NEIN**. Ihre Daten werden:
- NICHT verkauft
- NICHT an Werbetreibende weitergegeben
- NICHT mit Dritten geteilt

**Ausnahme**: Technischer Dienstleister:
- **Supabase** (EU): Für Datenspeicherung - arbeitet nur in unserem Auftrag

## KI-Funktionen

Die App nutzt KI für Gesundheitsempfehlungen.
- Anfragen werden über sichere Server verarbeitet
